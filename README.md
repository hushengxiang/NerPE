## [NeurIPS'24] Continuous Heatmap Regression for Pose Estimation via Implicit Neural Representation
![xxx](figures/overview.svg)
By introducing implicit neural representation (INR), the proposed NerPE achieves continuous heatmap regression for 2D human pose estimation. Thanks to the decoupling of INR and spatial resolution, NerPE can output the
predicted heatmaps at arbitrary resolutions.
